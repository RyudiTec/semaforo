Este projeto simula o funcionamento de um semáforo usando HTML e JavaScript, desenvolvido com base em um vídeo do YouTube.

No HTML, há uma imagem do semáforo e botões para cada cor (vermelho, amarelo, verde e automático).
No JavaScript, o código acessa esses elementos e troca a imagem do semáforo conforme o botão clicado.

No modo automático, é usado o setInterval para alternar as cores sozinho, seguindo a sequência correta.

Esse projeto me ajudou a aprender sobre eventos, funções, troca de imagens e uso de temporizadores em JavaScript.
